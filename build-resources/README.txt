Restaurant Management System
============================

Thank you for installing Restaurant Manager!

GETTING STARTED:
- Launch the application from your desktop or start menu
- The system will automatically start the local server
- Access different interfaces using the View menu or keyboard shortcuts:
  * Ctrl+1: POS Terminal
  * Ctrl+2: Kitchen Display
  * Ctrl+3: Admin Panel

SYSTEM REQUIREMENTS:
- Windows 7 or later / macOS 10.15+ / Linux with GTK3
- 4GB RAM minimum, 8GB recommended
- 500MB free disk space
- Network connection for real-time synchronization

FEATURES:
- Complete Point of Sale system
- Kitchen Display System for order management  
- Real-time customer order monitoring
- Administrative analytics and reporting
- Multi-device synchronization via WebSocket
- Offline capability with automatic sync
- Receipt printing support
- Barcode scanning integration

NETWORK DISCOVERY:
This application uses UDP port 44201 for automatic discovery of other Restaurant Manager instances on your network. Make sure this port is allowed through your firewall for multi-station setups.

PRIVACY:
Restaurant Manager operates locally on your system and network. 
No data is sent to external servers unless explicitly configured.
Your restaurant data remains private and under your control.
